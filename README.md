# Sudoku Solver.

Program that accepts the hints for a sudoku problem, and prints the solution to the screen.

## Getting Started

* Fill the array `sudoku_board` in the file `src/sudoku_solver.c` with the hints.
* `make compile`
* `. sudokusolver.out`
